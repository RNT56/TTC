# ForgedTTC validator v0.1.0

This first trusted-core candidate packages the Apache-2.0 `forge-validate` command
for Linux x86_64, macOS x86_64, and Windows x86_64, plus the
`@forge/validate-wasm` facade. It validates ModelSpec 2.1.0 and exposes active data
contract versions through `forge-validate version --json` and WASM `version()`.

Compatibility: validator report 1.x, replay 1.x plus the deprecated `replay.v1`
read alias, and EnvSpec 1.x. See [`../COMPATIBILITY.md`](../COMPATIBILITY.md).

This release does not include FORGE Desktop, live providers, hosted operations, or
field-proven hardware claims. Verify `SHA256SUMS` before use. Rollback is removal of
the installed binary/package; no database migration is included.
